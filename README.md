# Custom-System-Utils
my custom made system utilities for everyday use and operations

## run
my custom + fav + most used util for running python and c files using shorter arguement sizes
### for python;

```
run file.py n   // runs it normally
run file.py i   // runs it wiht interactive mode
```

<img width="501" height="214" alt="image" src="https://github.com/user-attachments/assets/037b5137-ea39-4f0e-8306-bacdf5c97744" />

### for C:
``` 
run file.c n   // compiles with normal gcc on the file
run file.c d   // compiles with debug information on the file
run file.c f   // compiles with fast optimizations ( -O3 ) enabled
```

## Close
simple util which logs you out of the system, helpful for not having to wait for a long CLI shutdown procedure and also keeps the system on sleep.
